# Smart-RAG-Chatbot

**A smart website chatbot using Retrieval-Augmented Generation (RAG) with n8n workflows, OpenAI embeddings, and a vector database for intelligent, real-time responses.**

---

## Features

- Automated content extraction from websites and documents.
- Semantic search using embeddings for accurate context-aware answers.
- Real-time conversational AI integration for website chat.
- Easy API endpoint for integrating the chatbot on any website.
- Scalable architecture using n8n workflows and vector databases.
- Supports multi-source documents and metadata filtering.

---

## Tech Stack

- **n8n Workflows** – for automation and orchestration.
- **OpenAI Embeddings** – for semantic search and knowledge representation.
- **Vector Database** – for storing and querying embeddings efficiently.
- **Python / Node.js** – for scripts and custom logic.
- **HTML/CSS/JS** – optional for website integration.
